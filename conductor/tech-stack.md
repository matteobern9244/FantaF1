# Tech Stack

## Frontend
- **React 18:** Libreria UI moderna per un'architettura basata su componenti.
- **TypeScript:** Garantisce la sicurezza dei tipi e una migliore esperienza di sviluppo in tutta l'interfaccia.
- **Vite:** Strumentazione frontend di nuova generazione per uno sviluppo e una build veloci.
- **Lucide React:** Icone per un'esperienza visiva coerente e leggera.

## Backend (Node.js - Principale)
- **Node.js:** Runtime JavaScript per servizi backend scalabili.
- **Express 5:** Web framework veloce e non opinionated per Node.js.
- **Mongoose:** Object Data Modeling (ODM) per MongoDB e Node.js.
- **CORS e dotenv:** Utility essenziali per servizi web sicuri e configurabili.

## Backend (C# - Porting/Parallelo)
- **ASP.NET Core:** Framework ad alte prestazioni per la creazione di app moderne, abilitate per il cloud e connesse a Internet.
- **Entity Framework Core:** Utilizzato per le interazioni con il database nel contesto C# (come indicato dai test di integrazione).

## Database e Infrastruttura
- **MongoDB:** Database di documenti NoSQL per un'archiviazione dei dati flessibile e scalabile.
- **Docker:** Containerizzazione per ambienti di sviluppo e deploy coerenti.
- **Render:** Piattaforma cloud per il deploy automatizzato e l'hosting.

## Quality Assurance e Test
- **Vitest:** Runner per test unitari e di integrazione estremamente veloce per lo stack Node/React.
- **Playwright:** Test end-to-end affidabili per app web moderne, utilizzato qui per la validazione della UI responsive.
- **Supertest:** Astrazione di alto livello per il test di HTTP, utilizzato per i test di integrazione API.
- **Coverlet:** Framework di code coverage cross-platform per .NET, utilizzato per il backend C#.
