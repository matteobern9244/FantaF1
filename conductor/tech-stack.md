# Tech Stack

## Frontend

- **React 18:** Libreria UI moderna per un'architettura basata su componenti.
- **TypeScript:** Garantisce la sicurezza dei tipi e una migliore esperienza di
  sviluppo in tutta l'interfaccia.
- **Vite:** Strumentazione frontend di nuova generazione per uno sviluppo e una
  build veloci.
- **Lucide React:** Icone per un'esperienza visiva coerente e leggera.

## Backend (C#)

- **.NET 10 / ASP.NET Core:** Framework ad alte prestazioni per la creazione di
  app moderne e backend API.
- **MongoDB.Driver:** Driver ufficiale C# per le interazioni dirette con il
  database MongoDB.

## Database e Infrastruttura

- **MongoDB:** Database di documenti NoSQL per un'archiviazione dei dati
  flessibile e scalabile.
- **Docker:** Containerizzazione per ambienti di sviluppo e deploy coerenti.
- **Render:** Piattaforma cloud per il deploy automatizzato e l'hosting.

## Quality Assurance e Test

- **Vitest:** Runner per test unitari ed End-to-End estremamente veloce per lo
  stack React.
- **Playwright:** Test end-to-end affidabili per app web moderne, utilizzato qui
  per la validazione della UI responsive.
- **Coverlet:** Framework di code coverage cross-platform per .NET, utilizzato
  per il backend C#.
